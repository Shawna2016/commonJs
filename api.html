数据绑定
$("xxx").fit() 如果返回的数据不用经过处理直接可以应用到实例中，即可直接在html标签中添加url属性，表示$.("xxx")范围内的请求会自动请求并且会自动识别标签进行数据绑定。
例如： <div class="container"><ul class="nav nav-pills" url="js/mock/jsonData.json"></ul></div>
绑定后:<div class="container"><ul class="nav nav-pills" url="js/mock/jsonData.json"> <li>HOME</li> <li>CSS3</li> <li>Sass</li> <li>jQuery</li> </ul></div>
$("xxx").ajaxData(params) 参数名和$.ajax的参数名相同。默认属性
	{
            url: "",
            data: "",
            type: "GET",
            async: false, 
            cache:true,
            dataType: "json", 
            contentType: "application/json",
            success: null, 
            error: null
	}
发送请求 例如：
$("#order").ajaxData({
        url: "js/mock/custormerOrder.json",
        type:"GET",
        dataType:"",
        success: function(data) {
           
        },
        error:function(){

    	}
});
$("xxx").setData(data) data:请求后处理好的数据 如果数据需要进行处理后才可以进行绑定，即可使用setData(data).
例如
$("#custormerMessage").ajaxData({
        url: "js/mock/custormerInfo.json",
        success: function(data) {
            var data = data.msg;
            $("#custormerMessage").setData(data);
        }
   });
$("xxx").load(params) params:string或object,发送请求的条件 加载数据。例如
$("#orderStatus").load("js/mock/orderStatus.json");
$("#orderStatus").load({
	url:"js/mock/orderStatus.json",
	data:data,
	type:"Post",
	success:function(){

	}
});
$("xxx").reload() 重新加载数据，使用load()后才能使用reload()。例如
$("#orderStatus").load("js/mock/orderStatus.json");
$("#orderStatus").reload();
表格操作
$("xxx").setGetCell([row,col,val]) Array
row: number/string 获取/设置行数
col：number/string 获取/设置列数
val: number/string 要设置的单元格值		 获取/设置单元格值
$("#tableA").setGetCell([2,3]);
$("#tableA").setGetCell([2,3,"caldfdgfd"]);
$("xxx").mergeCells(params) params:
rowIndex:第几行
colIndex:第几列
rowSpan:合并几行
colSpan:合并几列
[
	{ rowIndex: 2, colIndex: 2, rowSpan: 2, colSpan: 2 },
	{ rowIndex: 1, colIndex: 3, rowSpan: 3, colSpan: 3 }
 ]
 合并单元格
var cells = [
    { rowIndex: 2, colIndex: 2, rowSpan: 2, colSpan: 2 },
    { rowIndex: 1, colIndex: 3, rowSpan: 3, colSpan: 3 }
 ];
   $("#tableA").mergeCells(cells);
$("xxx").removeRows(params) params:string/number/array 删除一行或多行
$("#tableA").removeRows(2);
$("#tableA").removeRows([1,3,4]);
$("xxx").addRows(rowIndex,rows) rowIndex:在哪行后面增加
rows：增加的行数		 增加行
$("#tableA").addRows(1,2);
$("xxx").moveUpRows({rowIndex:x,rows:x,toRows:x}) rowIndex:第几行开始
rows:要移动rowIndex下面的几行
toRows：向上移动几行		 向上移动行
$("#tableA").moveUpRows({rowIndex:2,rows:2,toRows:1});
$("xxx").moveDownRows({rowIndex:x,rows:x,toRows:x}) rowIndex:第几行开始
rows:要移动rowIndex下面的几行
toRows：向下移动几行		 向下移动行
$("#tableA").moveDownRows({rowIndex:0,rows:2,toRows:1});
分页插件
$('.M-box1').pagination({
    totalData:100,
    showData:5,
    jump:true,
    coping:true,
    homePage:'首页',
    endPage:'末页',
    prevContent:'上页',
    nextContent:'下页'
});
 options(参数设置)
参数 默认值 说明
pageCount 9 总页数
totalData 0 数据总条
current 1 当前页数
showData 0 每页显示的条数
prevCls 'prev' 上一页class
nextCls 'next' 下一页class
prevContent '\<' 上一页节点内容
nextContent '>' 下一页节点内容
activeCls 'active' 当前页选中状态class名
count 3 当前选中页前后页数
coping false 是否开启首页和末页，值为boolean
homePage "" 首页节点内容，默认为空
endPage "" 尾页节点内容，默认为空
jump false 是否开启跳转到指定页数，值为boolean类型
jumpIptCls 'jump-ipt' 文本框内容
jumpBtnCls 'jump-btn' 跳转按钮class
jumpBtn '跳转' 跳转按钮文本内容

省市区三级联动插件
<div class="address">
	省：<select id="cmbProvince"></select>  
	市：<select id="cmbCity"></select>  
	区：<select id="cmbArea"></select>  
</div>
$(".address").ajaxData({
        url: "js/mock/provinceJson.json",
        success: function(data) {
          $(".address").address({
            provinceId:"cmbProvince",
            cityId:"cmbCity",
            areaId:"cmbArea",
            defaultProvince:"山东",
            defaultCity:"青岛",
            defaultArea:"城阳区",
            data:data
          });
        }
    });
 options(参数设置)
参数 默认值 说明
provinceId "" 省份selected id
cityId "" 市selected id
areaId "" 区selected id
defaultProvince "北京" 省份默认值
defaultCity '北京' 城市默认值
defaultArea '东城区' 区默认值
data "" ajax请求返回的数据
